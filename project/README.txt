Please start the program by running the mealplanner_main.py file.

When inputting the food would like to eat for each meal. Please be sure to follow the basic food structure below:

Breakfast: one beverage, one meat/diary, one vegetable, one fruit
Lunch: one beverage, one meat/seafood, one main, one vegetable
Dinner: one meat/seafood, one main, one vegetable, one fruit
